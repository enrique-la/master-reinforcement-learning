# Reinforcement Learning — Máster en IA & Data Science

Asignatura de aprendizaje por refuerzo: fundamentos teóricos, algoritmos clásicos y aplicaciones en IA de videojuegos.

## Contenido

### Fundamentos
- Procesos de Decisión de Markov (MDP)
- Políticas, recompensas y funciones de valor
- Exploración vs. explotación

### Algoritmos
- Q-Learning y Deep Q-Networks (DQN)
- Policy Gradient methods
- Actor-Critic (A2C/A3C)

### Aplicaciones
- Agentes para juegos y entornos simulados
- Toma de decisiones secuencial
